# Finance Committee Analytics & Sponsor Platform

A full-stack web platform to manage, analyze, and optimize a college Finance Committee's sponsor ecosystem and event finances.

Short summary:
- Track sponsors & partnerships
- Manage event budgets
- Analyze ROI, footfall & KPIs
- Generate sponsor proposals & reports
- Public portal, sponsor portal, internal dashboard

---

## Tech stack
- Frontend: HTML, CSS, JavaScript, Chart.js
- Backend: Python (Flask)
- Database: MySQL
- Utilities: PDF/Excel generation libraries

## Quickstart

Prerequisites:
- Python 3.8+
- MySQL server (local or remote)

1. Clone
    git clone <repo-url>
2. Backend
    cd finance_committee_platform/backend
    python -m venv venv
    source venv/bin/activate   # Windows: venv\Scripts\activate
    pip install -r requirements.txt
    cp ../.env.example .env    # edit .env with DB credentials and secrets
    # initialize DB (example)
    mysql -u USER -p DB_NAME < ../database/init.sql
    # run
    export FLASK_APP=app.py
    flask run --host=0.0.0.0   # Windows: set FLASK_APP=app.py & flask run
3. Frontend
    cd ../frontend
    # open index.html in browser or run a static server:
    python -m http.server 8000

Ports and endpoints can be configured in .env.
